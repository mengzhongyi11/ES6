## 箭头函数

**箭头函数（=>） 是 ES6 简洁函数写法，核心优化语法冗余，同时改变  this  绑定规则。**

1. ### 基本语法（简化 function 定义）

- 无参数： () => 表达式 

- 单参数： x => 表达式 （可省略括号）

- 多参数/代码块： (x, y) => { 语句; return 结果 } （代码块需显式 return）
// 普通函数 vs 箭头函数
const add1 = function(a, b) { return a + b; };
const add2 = (a, b) => a + b; // 简洁等价

2. ### 核心特性（关键区别于普通函数）

-  this  绑定：继承定义时的外层上下文 this，而非调用时的上下文（无自己的 this），无法通过 call/apply/bind 修改。

- 无 prototype：不能作为构造函数（不能用 new 实例化）。

- 不绑定 arguments：需用剩余参数（...args）获取参数列表。

3. ### 适用场景

- 回调函数（如数组方法、Promise.then）：简化代码，避免 this 指向混乱。

- 简单逻辑函数：无需复杂 this 操作的场景。

  ```js
  // 数组回调中简化
    const arr = [1,2,3];
    const doubled = arr.map(item => item * 2); // [2,4,6]
  
  // 避免 this 丢失（对比普通函数）
  const obj = {
    name: "Tom",
    sayHi: () => console.log(`Hi ${this.name}`) // this 继承外层（非 obj）
  };
  ```

  this代指的是传入变量的对象或在同一块作用域下的对象
